# Spring Boot Contact Form (Java)

This project demonstrates a basic Spring Boot backend that handles
HTML form submissions and prints submitted data to the console.

## Features
1. Spring Boot REST Controller
2. Handles POST form submission
3. No database integration (console output only)

## Tech Stack
1. Java
2. Spring Boot
3. Maven
4. HTML + CSS

## How to Run
1. Open project in VS Code
2. Run `ContactformApplication.java`
3. Open HTML file in browser
4. Submit the form
5. Check console output
